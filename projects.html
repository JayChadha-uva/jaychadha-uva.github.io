UVA Schedule Builder

August - December 2022

Using Django a web application was constructed enabling users to browse courses in the UVA course catalog and create schedules saved to their profiles. Course data was fetched from a UVA API. Data such as schedules, friends, and profiles were stored in a postgres database deployed on Heroku.
